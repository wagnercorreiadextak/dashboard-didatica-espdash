**Documentação dos tópicos**

**Dados dos sensores:**

Tópico: espdash/automacao/sensor
Objeto retornado: 

{
  "sensor": "temperatura",
  "valor": 23.78,
}

Legenda
sensor -> nome do sensor
valor -> valor coletado

**Controle dos relés:**

Tópico: espdash/automacao/rele
Objetos esperado:

{
  "rele": 0,
  "status": 1
}

Legenda
rele -> posição do relé
status\ -> 1: ligar / 2: desligar
